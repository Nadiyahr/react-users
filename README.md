## Techstack

* React.js v18
* Redux v4
* Redux Toolkit
* React Router v6
* Typescript
* SCSS preprocesor
* Bootstrap

### Dependencies

* Node v14.18.0 and higher
* NPM v6.14.15 and higher

### Installing

- Fork and clone this repository
- Run `npm install` in your terminal
- Run `npm start`
